Automated Testing Framework

Project Overview

This project provides a full‑stack testing framework ensuring data integrity across web applications, APIs, and data pipelines. It includes Selenium, Cypress, Playwright, and pytest suites with CI/CD integration.

Key Features

  • Multi‑framework testing (Selenium, Cypress, Playwright)
  • Data validation and quality checks
  • API testing and performance benchmarks
  • CI/CD pipelines with GitHub Actions
  • HTML reports with screenshots (Allure)
